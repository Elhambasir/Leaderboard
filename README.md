# 📖 To Do List <a name="about-project"></a>

The leaderboard website displays scores submitted by different players. It also allows you to submit your score. All data is preserved thanks to the external Leaderboard API service. In this activity I set up a JavaScript project for the Leaderboard list app, using webpack and ES6 features, notably modules. I developed a first working version of the app following a wireframe, but without styling - just focus on functionality. In following activities, I will consume the Leaderboard API using JavaScript async and await and add some styling.

<!-- > Describe your project in 1 or 2 sentences. -->
### How to build the "Leaderboard" 
- 1: Manage the scores (plain JS with objects).
- 2: Shows list of names and its scores.
- 3: Create a complete List of names and scores .

- git version 2.38.x
- node.js version > 12.x
- IDE
